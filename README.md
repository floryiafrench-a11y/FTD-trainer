# FTD Trainer for GitHub Pages

Готовый статический тренажёр по вкладке **«СВОД»** из файла **«Вопросы FTD_Апрель.xlsx»**.

## Что внутри
- `index.html` — интерфейс тренажёра
- `questions.js` — банк вопросов
- `.nojekyll` — чтобы GitHub Pages не применял Jekyll-обработку
- `build_questions_from_excel.py` — необязательный скрипт для пересборки `questions.js` из нового Excel

## Как опубликовать через GitHub Pages
1. Загрузите содержимое этой папки в репозиторий GitHub.
2. Откройте `Settings` → `Pages`.
3. В блоке `Build and deployment` выберите:
   - `Deploy from a branch`
   - ветка: `main`
   - папка: `/ (root)`
4. Сохраните настройки.
5. GitHub выдаст одну постоянную ссылку на `index.html`.

## Как обновлять вопросы, не меняя ссылку
Если путь к `index.html` остаётся тем же, ссылка на тренажёр остаётся прежней.

Есть два варианта обновления:
1. **Простой:** заменить файл `questions.js` на новый.
2. **Через скрипт:** положить рядом новый Excel и запустить:
   ```bash
   python build_questions_from_excel.py "Вопросы FTD_Апрель.xlsx"
   ```

После этого сделать `commit` и `push` в тот же репозиторий.

## Технические заметки
- В тренажёре используется банк из `159` вопросов.
- Вопросы взяты только со вкладки **«СВОД»**.
- Колонка **ID** в интерфейсе не используется.
- Прогресс хранится локально в браузере пользователя.
